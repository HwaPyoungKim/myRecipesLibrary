# 🍳 My Recipes Library

A polished personal recipe app designed to organize, preview, and share home-cooked dishes in a modern and user-friendly interface.

This project solves the problem of scattered recipe notes by turning recipes into a searchable, visually structured library with clear steps, ingredients, and cooking guidance.

## ✨ Features

- 🏠 Landing page with featured recipes and category highlights
- 🔍 Search recipes by title or ingredient
- 🎯 Filter recipes by category, difficulty, and preparation time
- 📋 Recipe detail pages with:
  - Ingredient lists
  - Step-by-step instructions
  - Cooking time, servings, and difficulty
  - Personal notes or tips
- 📱 Fully responsive design for mobile, tablet, and desktop
- 🎨 Clean, modern UI built for readability and ease of use

---

## 🛠 Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Data:** Static JSON / mock recipe data

---

## 🧠 Architecture & Decisions

- Built with a **component-driven architecture** for reusability and easier scaling.
- Uses **static recipe data** to focus on frontend polish, performance, and UX.
- Designed as a **frontend-first MVP**, ready to connect to a backend service like Supabase, Firebase, or a custom API.
- Mobile-first layout ensures a great experience on small screens without sacrificing desktop clarity.

