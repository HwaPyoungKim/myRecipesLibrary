# CONTEXT

## Qué hace

`src/assets` contiene imágenes JPG usadas por la interfaz.

## Qué problema resuelve

Entrega assets visuales versionados con el código y procesados por Vite para generar rutas finales con hash.

## Cómo usarlo

- Agregar la imagen en `src/assets`.
- Importarla desde una ruta o desde `src/data/recipes.ts`.
- Usarla como `src` de un `<img>`.
- Mantener nombres descriptivos y estables.
