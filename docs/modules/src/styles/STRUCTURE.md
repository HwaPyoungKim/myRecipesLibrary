# STRUCTURE

## Responsabilidad

Definir el sistema visual global.

## Estructura

```text
src/styles.css
├─ imports Tailwind y animaciones
├─ custom variant dark
├─ @theme inline
├─ variables :root
├─ variables .dark
└─ base layer
```

## Notas

- Los headings usan `font-display`.
- El body usa `font-sans`.
- `@source "../src"` permite a Tailwind escanear el código fuente.
