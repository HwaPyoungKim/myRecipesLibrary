# CONTEXT

## Qué hace

`src/styles.css` define los estilos globales, imports Tailwind, tokens de tema, tipografías y variables CSS.

## Qué problema resuelve

Mantiene una identidad visual consistente para toda la aplicación sin repetir colores, sombras y fuentes en cada componente.

## Cómo usarlo

- Ajustar tokens en `:root` para cambiar tema claro.
- Ajustar `.dark` para tema oscuro.
- Usar variables mediante clases Tailwind como `bg-background`, `text-foreground` y `text-primary`.
- Mantener imports Tailwind al inicio.

## Tema actual

Usa base crema cálida, acento terracota, soporte sage, tipografía display `Playfair Display` y sans `Inter`.
